Tema 1 - Texto
Propiedad Valor Inicial Descripción
color initial Define el color del texto.
font-family initial Especifica la familia tipográfica del texto.
font-size 16px Especifica el tamaño de la fuente.
font-weight normal Define el grosor de la fuente.
font-style normal Especifica el estilo de la fuente, como normal, cursiva o oblicua.
font-variant normal Controla la forma de la fuente, como minúsculas pequeñas (small-caps).
font initial Define todas las propiedades de la fuente en una sola declaración.
text-align left Define la alineación del texto (izquierda, derecha, centro o justificado).
line-height normal Establece la altura de la línea, es decir, el espacio entre líneas de texto.
text-decoration none Especifica la decoración del texto, como subrayado, tachado, etc.
text-transform none Modifica el caso del texto, como convertir a mayúsculas o minúsculas.
vertical-align baseline Controla la alineación vertical del texto en relación con su contenedor.
text-indent 0 Define la cantidad de sangría en el primer renglón de un párrafo.
letter-spacing normal Controla el espacio entre las letras de un texto.
word-spacing normal Controla el espacio entre las palabras en un texto.
white-space normal Controla cómo se gestionan los espacios en blanco dentro del texto.
Tema 2 - Enlaces
Característica Propiedades Descripción
Tamaño, Color y Decoración color, font-size, text-decoration Define el color, tamaño y la decoración del enlace (por ejemplo, subrayado o sin subrayado).
Decoración Personalizada border, padding, background-color Permite personalizar la apariencia del enlace con bordes, relleno y fondo.
Tema 3 - Imágenes
Característica Propiedades Descripción
Anchura y Altura width, height Define las dimensiones de la imagen (anchura y altura).
Eliminar el Borde border Elimina el borde alrededor de la imagen, si es necesario.
Tema 4 - Listas
Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador para la lista, como un círculo (disc), cuadrado (square) o número (decimal).
list-style-position outside Define si el marcador de lista está dentro o fuera del cuadro de la lista.
list-style-image none Permite usar una imagen como marcador de lista en lugar de un marcador estándar.
list-style list-style-type, list-style-position, list-style-image Propiedad abreviada que combina `list-style-type`, `list-style-position` y `list-style-image`.
Tema 5 - Tablas
Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table separate Controla si los bordes de las celdas deben colapsar en un solo borde o permanecer separados.
border-spacing table 0 Establece el espacio entre las celdas de una tabla.
empty-cells table show Define si las celdas vacías deben ser visibles o no.
